##命令行-第四章-操作文件与目录
1.通配符
    
    *                匹配任意多个字符（包括0个和1个）
    ？               匹配任意单个字符（不包括0个）
    [characters]    匹配任意一个属于字符集中的字符
    [!characters]   匹配任意一个不属于字符集的字符
    [[:class:]]     匹配任意一个属于指定字符类中的字符

2.常用的字符类

    [:alnum:]      匹配任意一个字母或数字
    [:alpha:]       匹配任意一个字母
    [:digit:]       匹配任意一个数字
    [:lower:]       匹配任意个小写字符
    [:upper:]       匹配任意一个大写字符
    
3.通配符实例

    *           所有文件
    g*          以g开头的任一文件
    b*.txt      以b开头，中间有任意多个字符，并以.txt结尾的任一文件
    Data???     以Data开头，后面跟3个字符的任一文件
    [abc]*      以abc中的任一个开头的任一文件
    BACKUP.[0-9][0-9][0-9]  以BACKUP.开头 后面紧跟三个数字的任一文件
    [[:upper:]]*    以大写字母开头的任一文件
    [![:digit:]]*   不以数字开头的任一文件
    *[[:lower]123]  以小写字母或数字 1 2 3中的任一个结尾的任一文件
    
4.mkdir 创建目录
    
    mkdir dir1 创建单个目录
    mkdir dir1 dir2 dir3 ... 创建多个目录
    mkdir -m=777 dir1 创建目录dir1并设置权限777
    mkdir -p dir1/dir2/dir3/... 创建目录dir3 如果父级别目录不存在会自动创建

5.cp 复制文件和目录
    
    cp item1 item2  将单个文件或目录item1或者到文件或目录item2中
    cp -a item1 item2 复制文件和目录极其属性,包括所有权限
    cp -i item1 item2 在覆盖一个已存在的文件前，提示用进行确认是否要覆盖
    cp -r item1 item2 递归地复制目录极其内容
    cp -u item1 item2 更新复制。复制一个目录时，只会复制不存在的文件，或是更新文件
    
6.mv 移除和重命名文件
    
    mv item1 item2 将item1移动到item2
    mv -i item1 item2 如果item2已经存在 会提示是否要覆盖
    mv -u item1 item2 只移动item2不存在的文件或者更新文件
    
7.rm 删除文件和目录
        
    rm item 删除item文件或文件夹 
    rm -i item      删除前提示用户确认
    rm -r item      递归删除目录
    rm -f item      强制删除
    rm -v item      删除文件时显示信息性消息
    
8.rmdir 删除目录
    
    rmdir dir1 删除目录dir1
    rmdir -p dir1/dir2/dir3 删除所有子目录
    
9.ln 硬链接&软链接
    
    ln filename filename-hard 创建filename-hard硬链接
